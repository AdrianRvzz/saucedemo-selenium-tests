# Caso de prueba: Logout

## Objetivo
Validar que el usuario pueda cerrar sesión correctamente.

## Script
scripts/test_logout.py

## Pasos ejecutados
1. Login con usuario standard_user.
2. Abrir menú lateral.
3. Hacer click en "Logout".
4. Validar que el usuario regrese a la página de login.
5. Guardar captura de pantalla.

## Evidencias generadas
- Captura: evidencias/logout/logout_exitoso.png
- Log: evidencias/logout/log_YYYYMMDD_HHMMSS.txt

## Resultados
- Logout exitoso y usuario regresó a login.
